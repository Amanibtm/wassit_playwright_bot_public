# WassitPlaywrightBot: Automated Grant Date Checker

Reliable browser bot that monitors the Wassit unemployment grant website every 5–20 minutes and notifies the user the moment dates become available.

**Features**
- Playwright browser automation
- Simple Flet user interface for configuration
- Real-time notifications for high-competition appointments

**Technologies**
Python • Playwright • Flet

**Status**  
Personal project — core functionality working. Paused due to competing solutions.
